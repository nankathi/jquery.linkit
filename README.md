#LinkIt jquery Plugin#

**LinkIt** is a simple jQuery plugin that allows you to attach a link to any elements

##Versions##
0.1.0 Inital
0.2.0 Updated

##Usage##
        $(document).ready(function(){
                $('span').linkIt({
                    //whatever : 'anything'
                    href : 'http:test.com',
                    text: 'Click Me',
                    target : '_blank'
                });
        });
